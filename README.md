# NEVDEX

Minimal TON swap frontend powered by STON.fi market data and TonConnect.

## Stack

- Vite
- Vanilla JavaScript
- STON.fi API/SDK
- TonConnect UI

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint:frontend
```

## Environment

Copy `.env.example` to `.env` if you need custom network, RPC, token, or TonConnect manifest settings.
